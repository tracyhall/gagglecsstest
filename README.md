# gagglecsstest
Basic CSS Test for Gaggle applicants.

- No widths or heights should be set in pixels, with the exception of ".container".

- No mobile styles should be inside of media queries.

- Colors in screenshots: 
-- Orange: #F26722;
-- Blue: #263692;
-- Gray: #666666;
-- Light Gray: #eeeeee;
